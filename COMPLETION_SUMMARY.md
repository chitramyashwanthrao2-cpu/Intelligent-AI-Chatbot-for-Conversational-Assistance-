## 🐛 Troubleshooting

### Error: Cannot connect to server

**Solution:**

```bash
# Make sure backend is running
python run_backend.py

# Check if port 5000 is available
netstat -ano | findstr :5000
```

### Error: Styles not loading

**Solution:**

- Make sure `style.css` is in `frontend/` folder
- Verify file path in index.html
- Check CSS file isn't corrupted

### Error: JavaScript not working

**Solution:**

- Make sure `script.js` is in `frontend/` folder
- Open browser console (F12)
- Look for JavaScript errors

### Message history not saved

**Solution:**

- Enable localStorage in browser
- Check browser privacy settings
- Try different browser
